# Vi Fit

A neighbourhood gym in Kelowna focused on creating a welcoming, non-intimidating fitness environment.

**Website:** https://vifit.ca/

## Overview

Vi Fit is a boutique gym designed for people who feel intimidated by traditional gym culture. The gym caps memberships to maintain a calm, comfortable atmosphere and provides personalized support for beginners.
